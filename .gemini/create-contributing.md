# Prompt: Creación de Guía de Contribución

**ACTÚA COMO:** Antigravity Gemini 3 Flash, lead de documentación y procesos.

**OBJETIVO:**
Generar o actualizar `CONTRIBUTING.md` en la raíz del repositorio, alineado con **TASK-013** y el estándar documental del proyecto.

**INSTRUCCIONES:**

1. **Estructura Requerida:**
   - Banner de bienvenida y alineación con `[TASK.MD](./docs/TASK.MD)` y `[PROJECT_STRUCTURE.md](./docs/PROJECT_STRUCTURE.md)`.
   - Flujo de Git (Trunk-Based / Branch naming).
   - Convención de Commits (Conventional Commits).
   - Estándares de Código (TypeScript estricto, R3F, Barrel Files, uso de public.ts).
   - Proceso de Pull Request y Definition of Done (DoD).
   - Link a `README.md` para setup y a `PROJECT_STRUCTURE.md` para arquitectura.
2. **Contexto:**
   - El documento debe ser motivador pero técnico, claro para cualquier futuro colaborador.
   - Incluir ejemplos de nombres de ramas, commits y PRs.
   - Aclarar que toda contribución debe referenciar una TASK-ID y actualizar la documentación relevante.
3. **Footer:**
   - Incluir referencia a `[TASK.MD](./docs/TASK.MD)` y fecha de última actualización.

**SALIDA ESPERADA:**
Un manual claro, profesional y accionable para que cualquier futuro colaborador pueda integrarse al proyecto sin fricciones y manteniendo la calidad y la coherencia arquitectónica.
