# Prompt: Creación de Guía de Contribución

**ACTÚA COMO:** Antigravity Gemini 3 Flash, lead de documentación y procesos.

**OBJETIVO:**
Generar o actualizar `CONTRIBUTING.md` en la raíz del repositorio, alineado con la **TASK-013**.

**INSTRUCCIONES:**

1. **Estructura Requerida:**
   - Flujo de Git (Trunk-Based / Branch naming).
   - Convención de Commits (Conventional Commits).
   - Estándares de Código (Typescript, R3F, Barrels).
   - Proceso de Pull Request y Definition of Done (DoD).
2. **Contexto:**
   - Debe ser motivador pero técnico.
   - Linkear a `PROJECT_STRUCTURE.md` como guía arquitectónica.

**SALIDA ESPERADA:**
Un manual claro para que cualquier futuro colaborador pueda integrarse al proyecto sin fricciones y manteniendo la calidad.
